# Upload this project to GitHub

Open a terminal inside this folder and run:

```bash
git init
git add .
git commit -m "Add Python for Cybersecurity companion code"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/python-for-cybersecurity-code.git
git push -u origin main
```

Create the empty repository on GitHub before running the last two commands. Replace `YOUR-USERNAME` with your GitHub username.
